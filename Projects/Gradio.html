Web App con Gradio

Questo progetto è una web application interattiva sviluppata utilizzando Gradio, una libreria Python che permette di creare interfacce web semplici e veloci per modelli di machine learning e funzioni Python.

Con Gradio, è possibile trasformare rapidamente una funzione Python in una web app accessibile via browser senza dover scrivere codice HTML, CSS o JavaScript. L'interfaccia è intuitiva e adatta anche per utenti non tecnici.

In questa web app ho integrato due applicazioni che consentono all'utente di dialogare con un Chatbox o di generare dei file audio attraverso il modello TTS di Kokoro.

Il progetto è stato deployato su Hugging Face Spaces ed è accessibile pubblicamente.
